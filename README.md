# jaker-dev.github.io
Portfolio for Jake

## Dependencies

You should be running:

- Node > v22.0
- If using NVM, you can run:

```bash
nvm use 24.7.0
```

## Get up and running

Clone the repo down to your local

Run from the project root:

```bash
npm i
```

To start building assets:

```bash
npm run dev
```

To build assets before deploying:

```bash
npm run build
```
